> Content adaptation for different brains. Screening (not diagnosis). Executive function support. IEP/504 plan assistance.

## The Solution

**ContentAdapter** — Reformat any text for dyslexia-friendly, ADHD-optimized chunking, autism visual scaffolding\n**ScreeningAgent** — Flag potential learning differences through reading/writing analysis (triage to professionals)\n**ExecutiveCoach** — Task decomposition, time estimation, transition warnings, priority coaching\n**IEPAssistant** — Help parents understand rights, draft accommodation requests\n**ToolFinder** — Free assistive technology matching by need

## Agents

This is a support tool, NOT a diagnostic device. Always consult qualified professionals for diagnosis.
